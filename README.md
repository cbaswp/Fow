# Fow - MIR4 SA31 Guide

Guía no oficial para el servidor SA31 de MIR4. Consulta horarios de bosses, planifica expediciones de clan y calcula recetas de forjado.

**Ver en vivo:** https://cbaswp.github.io/Fow

## Características

- **Horarios de Bosses** — Lista completa de bosses con horarios en hora de São Paulo
- **Expediciones de Clan** — Calculadora de puntos según miembros activos
- **Forjado (Crafting)** — Calculadora de materiales para crear armas, armaduras, cuernos y garras

## Tecnologías

HTML5 · CSS3 · JavaScript (ES6)

## Idioma

Español · Português · English

El idioma se guarda automáticamente en localStorage del navegador.
